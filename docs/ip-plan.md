# IP addressing plan

## Management network (192.168.100.0/24)

| IP              | Device                              |
| --------------- | ----------------------------------- |
| 192.168.100.1   | mgmt gateway (upstream or pfSense)  |
| 192.168.100.10  | jump host / Raspberry Pi            |
| 192.168.100.11  | sw1                                 |
| 192.168.100.12  | sw2                                 |
| 192.168.100.13  | sw3                                 |
| 192.168.100.21  | r1                                  |
| 192.168.100.22  | r2                                  |
| 192.168.100.23  | r3                                  |

## L3 point-to-point links (OSPF area 0)

Use `/30` subnets from `10.0.0.0/16`.

| Link     | Subnet           | A-side        | B-side        |
| -------- | ---------------- | ------------- | ------------- |
| r1 <-> r2 | 10.0.0.0/30     | r1 .1         | r2 .2         |
| r2 <-> r3 | 10.0.0.4/30     | r2 .5         | r3 .6         |
| r1 <-> r3 | 10.0.0.8/30     | r1 .9         | r3 .10        |

## LAN / user VLANs

| VLAN | Name     | Subnet          | Gateway       |
| ---- | -------- | --------------- | ------------- |
| 10   | USERS    | 10.10.10.0/24   | 10.10.10.1    |
| 20   | SERVERS  | 10.20.20.0/24   | 10.20.20.1    |
| 30   | VOICE    | 10.30.30.0/24   | 10.30.30.1    |
| 99   | MGMT     | 192.168.100.0/24| 192.168.100.1 |
| 999  | UNUSED   | (native trunk)  | n/a           |
